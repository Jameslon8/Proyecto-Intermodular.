\# Sistemas Informaticos - HuntPMO



En esta carpeta esta la documentación relacionada con el modulo de Sistemas Informáticos.



\## Archivos



\- `TRABAJO\_DOCUMENTACION\_SISTEMAS.odt`: trabajo sobre instalación, despliegue y funcionamiento de aplicaciones Java en entornos reales.



\## Contenido del trabajo



El documento de Sistemas explica:



\- Entorno de ejecución de aplicaciones Java

\- Creación de instaladores para aplicaciones Java

\- Servicios, APIs y comunicación en red

\-  Contenedores con Docker

\-  Aplicaciones frontend con Angular y React



\## Objetivo



El objetivo de esta parte es explicar que necesita una aplicación para poder ejecutarse en un entorno real, teniendo en cuenta el sistema operativo, Java, dependencias, configuración y herramientas de despliegue.
